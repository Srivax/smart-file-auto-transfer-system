 # Sprint 1 – File Upload & Validation

## Epic
File upload module allowing teachers to upload marks Excel files.

## User Stories
1️⃣ As a teacher, I should be able to upload Excel files (.xlsx) of marks.  
2️⃣ As a system, I should validate uploaded columns before storing.

## Tasks
✅ Design Upload API using Express and Multer  
✅ Parse Excel using XLSX library  
✅ Connect MongoDB Atlas and create Student schema  
✅ Validate data and handle errors  

## Acceptance Criteria
✔️ Uploaded file accepted successfully  
✔️ Data stored in MongoDB  
✔️ Invalid formats rejected
